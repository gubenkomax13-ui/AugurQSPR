# Детали связей

Эта таблица содержит все расчёты по связям между паттернами.

Каждая строка — сравнение двух структурных паттернов внутри одной молекулярной формулы.

Основные поля:

- **Связь паттернов** — какая пара сравнивается.
- **Формула** — внутри какой группы изомеров выполнено сравнение.
- **Значение A** и **Значение B** — исходные значения свойства.
- **Δ = A − B** — разность между паттернами.
- **Ожидаемая Δ** — значение, ожидаемое по найденному тренду.
- **ΔΔ** — изменение разности относительно предыдущей формулы.
- **Статус связи** — согласовано или правило ломается.
