\# Генетические и интерпретируемые методы



\## GEP Symbolic Regression



GEP — метод генетического программирования. Он может подбирать не только коэффициенты, но и вид математической формулы.



Преимущество — потенциально интерпретируемое уравнение модели.



В текущей версии используется лёгкая symbolic-regression реализация без внешних зависимостей.



\## CART Regression



CART — решающее дерево для регрессии.



Модель делит пространство дескрипторов на области и в каждой области даёт прогноз.



Плюс — простая интерпретация. Минус — риск переобучения без ограничения глубины.



\## MARS-like Regression



MARS-like Regression — приближённая реализация идеи кусочной регрессии через расширенные признаки и регуляризацию.



Может быть полезна для нелинейных, но относительно интерпретируемых зависимостей.
