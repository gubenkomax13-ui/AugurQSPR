\# Методика: спектр как дескриптор структуры



В этом режиме спектр рассматривается как числовое представление молекулярной структуры.



Для корректного сравнения веществ важно соблюдать три условия:



1\. одна и та же область спектра для всех веществ;

2\. одинаковый шаг сетки;

3\. сопоставимая нормировка интенсивности.



\## IR



Для IR-спектров желательно использовать сопоставимые фазы и условия регистрации. Для строгих QSPR-моделей лучше не смешивать gas, liquid, solid, KBr, film и solution без необходимости.



Близкий строгий режим:



\- gas-спектры;

\- область примерно 550–3798 см⁻¹;

\- единый шаг сетки;

\- GRID-дескрипторы;

\- SVD-компоненты.



\## Mass



Для масс-спектров используется сетка по m/z. Обычно применяют шаг 1 и нормировку по сумме, максимуму или векторной норме.



\## Типы спектральных дескрипторов



\- GRID — интенсивности на общей сетке;

\- BINARY — наличие/отсутствие сигнала;

\- BAND — признаки по областям спектра;

\- SVD — сжатое представление спектра;

\- PEAKS — признаки по пикам.
