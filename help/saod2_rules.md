# Найденные правила

Эта таблица показывает правила, которые программа сама нашла в данных.

Правило — это устойчивая связь между двумя структурными паттернами.

Например:

- `n-alkane minus 2-methyl`
- `2-methyl minus 3-methyl`
- `2-methyl; 2-methyl minus 2-methyl; 3-methyl`

Для каждой связи программа смотрит:

- сколько формул можно сравнить;
- кто обычно выше по значению свойства;
- насколько устойчив знак разности;
- насколько плавно меняется разность;
- есть ли поломки правила.

Важно: направление правила не задаётся заранее. Программа сама определяет, кто обычно выше или ниже для данного свойства.
