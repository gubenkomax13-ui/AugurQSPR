# Непроверяемые и слабо проверяемые вещества

Эта таблица показывает вещества, для которых в датасете мало внутренних аналогов.

Важно: непроверяемое вещество не является выбросом автоматически.

Оно означает:

- нет собственного ряда;
- мало изомеров той же формулы;
- нет устойчивых правил вокруг вещества.

Такие вещества могут зашумлять модель, потому что их экспериментальные значения трудно проверить внутренними закономерностями датасета.
