# Как выбрать режим отбора дескрипторов

## Быстрый отбор

Хороший режим по умолчанию.

Он удаляет константные признаки, удаляет сильно коррелирующие дескрипторы и оставляет top-k признаков по связи со свойством.

## F-regression

Оценивает линейную связь дескриптора с целевым свойством.

Подходит для простых и линейных моделей.

## Mutual Information

Ищет нелинейную связь между дескриптором и свойством.

Полезно для Random Forest, SVR, KNN и других нелинейных моделей.

## LASSO selection

Оставляет признаки с ненулевыми коэффициентами LASSO.

Подходит для разреженных линейных моделей.

## Random Forest importance

Отбор по важности признаков в Random Forest.

Подходит для нелинейных зависимостей, но может быть медленнее.

## RFE + Ridge

Рекурсивное удаление признаков.

Может быть полезно, но на тысячах дескрипторов работает медленнее.

## Рекомендуемый старт

- включить автоматический отбор;
- режим: быстрый отбор;
- порог корреляции: 0.95;
- оставить 20–100 дескрипторов;
- затем включить оптимизацию гиперпараметров.
