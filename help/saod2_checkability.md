# Проверяемость веществ

Эта таблица отвечает на вопрос: насколько значение свойства можно проверить внутренними закономерностями датасета.

Вещество может быть:

- проверяемым по собственному ряду;
- проверяемым по изомерам той же формулы;
- проверяемым через сеть найденных правил;
- уникальным по паттерну, но всё ещё проверяемым по формуле.

Важно:

**Уникальное по паттерну** не означает **непроверяемое вообще**.

Например вещество может иметь единственный точный паттерн замещения, но входить в большую группу изомеров C10H22. Тогда оно проверяемо внутри группы молекулярной формулы.

Основные поля:

- **Размер собственного ряда** — сколько веществ имеют такой же точный паттерн.
- **Размер группы изомеров формулы** — сколько веществ имеют ту же молекулярную формулу.
- **Проверяемость по собственному ряду** — есть ли гомологический ряд.
- **Проверяемость по формуле** — есть ли изомеры той же молекулярной массы.
- **Итоговая проверяемость** — общий вывод.
- **Эвристический индекс проверяемости** — ручная шкала, основанная на числе рядов,
  формульных групп и независимых структурных проверок.
- **Риск зашумления выборки** — насколько осторожно нужно использовать вещество в модели.

Индекс проверяемости не является вероятностью корректности записи. Это эвристическая
оценка того, насколько много внутренних химических сравнений доступно для вещества.
