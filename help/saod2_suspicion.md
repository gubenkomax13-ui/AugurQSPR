# Подозрительные вещества

Эта таблица агрегирует поломки правил на уровне вещества.

Если вещество участвует в нескольких сломанных связях, его подозрительность растёт.

Пример:

Если для вещества A ломаются связи:

- A ↔ B
- A ↔ C
- A ↔ D

а связи между B, C и D согласованы, то подозрение концентрируется на A.

Статусы:

- **согласовано** — вещество не ломает найденные правила;
- **требует проверки** — есть отдельные нарушения;
- **сильное структурное несоответствие** — несколько независимых структурных проверок не согласуются;
- **приоритетная ручная проверка** — множественные несогласованности или очень сильное отклонение;
- **почти непроверяемое** — мало связей для вывода.

Эта шкала является эвристической. Она помогает расставить приоритеты ручной проверки,
но не является статистически валидированной вероятностью ошибки.
