\# Ядерные методы и методы сходства



\## SVR



SVR — гибкий ядерный метод регрессии. Он может хорошо работать на небольших и средних выборках, особенно если зависимость между дескрипторами и свойством нелинейная.



\## Gaussian Process Regression



Gaussian Process Regression строит нелинейную регрессию и может оценивать неопределённость прогноза.



Это полезно для оценки надёжности QSPR-модели.



\## KNN Regression



KNN предсказывает свойство вещества по ближайшим веществам в пространстве дескрипторов.



Метод простой и понятный, но чувствителен к масштабу признаков и выбору числа соседей.
