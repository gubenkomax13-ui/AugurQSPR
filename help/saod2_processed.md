# Обработанные данные SAOD v2

Эта таблица показывает, как программа распознала каждое вещество по SMILES.

Важно: классификация выполняется не по названию вещества, а по молекулярному графу, построенному из SMILES.

Основные поля:

- **Канонический SMILES** — стандартизированное представление структуры.
- **Молекулярная формула** — рассчитана из структуры.
- **Число атомов углерода** — используется для сравнения гомологических рядов.
- **Точный паттерн замещения** — главный структурный признак для SAOD v2.
- **Индекс разветвлённости** — показывает, насколько структура отличается от нормальной цепи.
- **Конфликт дубликатов** — одна и та же структура имеет разные значения свойства.

Если вещество попало в неправильный паттерн, сначала проверьте SMILES, а не название.
