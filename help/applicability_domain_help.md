\# 🧭 Applicability Domain и Williams Plot



\## Что такое Applicability Domain (AD)



Applicability Domain (область применимости модели) показывает, насколько новое вещество похоже на вещества, использованные при обучении модели.



Даже очень точная модель может давать ненадёжные прогнозы для структур, сильно отличающихся от обучающей выборки.



\---



\## Что такое leverage h



Leverage показывает удалённость вещества от центра обучающей выборки в пространстве дескрипторов.



Чем больше leverage:



\* тем менее типичным является вещество;

\* тем выше риск экстраполяции;

\* тем менее надёжен прогноз.



\---



\## Порог h\*



Классический порог:



h\* = 3(p + 1) / n



где:



\* p — число дескрипторов модели;

\* n — число веществ в обучающей выборке.



Если:



h < h\*



вещество считается находящимся внутри области применимости модели.



Если:



h > h\*



вещество считается находящимся вне области применимости модели.



\---



\## Пример



Допустим, модель обучалась на алканах C1–C10.



Для декана leverage будет небольшим.



Для ароматического спирта leverage может оказаться очень высоким, поскольку такая структура отсутствовала в обучающей выборке.



\---



\# 📊 Leverage Distribution



График показывает распределение leverage для всех обучающих веществ.



Красная пунктирная линия соответствует порогу h\*.



Точки выше линии находятся вне области применимости модели.



\---



\# 📍 Williams Plot



Williams Plot одновременно показывает:



\* удалённость вещества от обучающей выборки;

\* величину ошибки модели.



Ось X:



Leverage h



Ось Y:



Стандартизованный остаток.



\---



\## Что такое стандартизованный остаток



Стандартизованный остаток показывает, насколько сильно модель ошиблась для конкретного вещества.



Значение около нуля означает хороший прогноз.



Чем дальше точка от нуля, тем больше ошибка.



\---



\## Большие остатки



Обычно используются границы:



+3

\-3



Если:



|остаток| > 3



вещество считается потенциальным выбросом.



\---



\## Критические точки



Критическая точка одновременно:



\* находится вне области применимости;

\* имеет большой остаток.



То есть:



h > h\*

и

|остаток| > 3



Это наиболее проблемные вещества для модели.



\---



\# Интерпретация Williams Plot



\## Внутри AD + маленький остаток



Модель работает корректно.



\## Внутри AD + большой остаток



Возможен экспериментальный выброс или ошибка данных.



\## Вне AD + маленький остаток



Редкая структура, прогноз следует интерпретировать осторожно.



\## Вне AD + большой остаток



Критическая точка.



Такие вещества требуют отдельного анализа.



\---



\# Как использовать в Augur QSPR



\## Вне AD = 0



Все обучающие вещества находятся внутри области применимости модели.



\## Большие остатки > 0



Следует проверить соответствующие структуры и экспериментальные данные.



\## Критические точки > 0



Необходимо внимательно анализировать эти вещества перед публикацией модели.
