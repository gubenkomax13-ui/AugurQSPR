# Applicability Domain и Williams Plot

**Applicability Domain (AD)** показывает, насколько вещество похоже на обучающую выборку в пространстве дескрипторов. Даже хорошая модель может ошибаться на структурах, которые сильно отличаются от обучающих объектов.

**Leverage h** отражает удалённость точки от центра обучающей выборки: чем выше `h`, тем выше риск экстраполяции. Классический порог: `h* = 3(p + 1) / n`, где `p` - число дескрипторов модели, `n` - число обучающих веществ. Если `h <= h*`, объект внутри AD; если `h > h*`, объект вне AD.

**Williams Plot** объединяет два признака риска: `h` по оси X и стандартизованный остаток по оси Y. Остаток около нуля означает хороший прогноз; обычно `|остаток| > 3` считают большим отклонением.

| Ситуация | Интерпретация |
|---|---|
| Внутри AD + маленький остаток | Модель работает штатно. |
| Внутри AD + большой остаток | Возможен экспериментальный выброс, ошибка данных или локальная слабость модели. |
| Вне AD + маленький остаток | Структура редкая; прогноз использовать осторожно. |
| Вне AD + большой остаток | Критическая точка, требует отдельной проверки. |

В Augur QSPR смотрите три показателя: **вне AD**, **большие остатки** и **критические точки**. Если критических точек нет, модель обычно стабильнее для текущей обучающей области; если они есть, проверьте соответствующие структуры, значения целевого свойства и дескрипторы.
